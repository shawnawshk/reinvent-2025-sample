# AWS re:Invent 2025 Sample Code

This repository contains sample code and demonstrations from AWS re:Invent 2025.

## What's Included

- **lambda-durable-functions** - Sample implementation showcasing durable execution patterns with AWS Lambda
- **lambda-managed-instances** - Demo of Lambda Managed Instances running functions on EC2 with serverless simplicity

Each sample includes its own documentation and deployment instructions.

## Purpose

These samples are provided as reference implementations to help you understand and implement serverless patterns and AWS services discussed at re:Invent 2025.
